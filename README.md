# 🚀 Học Next.js - Khóa "React Pro Max với Next.js"

Repository này lưu trữ toàn bộ source code, tài liệu và ghi chú cá nhân của tôi trong quá trình hoàn thành khóa học **"React Pro Max với Next.js"**. Mục tiêu của repo là theo dõi tiến độ học tập, lưu lại các kiến thức quan trọng và xây dựng dự án thực tế (SoundCloud Clone).

## 🛠 Tech Stack

- **Frontend:** React.js, Next.js (App Router / Pages Router), Material UI (MUI), Ant Design (Antd)
- **Kiến thức liên quan:** JavaScript cơ bản/nâng cao, React Hooks, Server Actions
- **Khác:** Chrome Dev Tools, Wavesurfer.js, SEO Optimization
